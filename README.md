# Guess The Number Game 🎯

## Internship Task
This project is developed as part of the **Java Programming Internship** at Oasis Infobyte.

## Project Description
Guess The Number Game is a simple console-based Java game where the computer generates a random number between **1 and 100**, and the user tries to guess it.

The program provides hints if the guessed number is **higher or lower** than the generated number. The game includes **limited attempts, multiple rounds, and a scoring system** to make it more interactive.

This project demonstrates the use of **Java programming concepts such as loops, conditionals, random number generation, and user input handling.**

---

## Features

- Random number generation (1–100)
- User guessing system
- Hints for higher or lower guesses
- Limited number of attempts
- Multiple rounds gameplay
- Score calculation based on attempts
- Total score tracking

---

## Game Rules

1. The system generates a random number between **1 and 100**.
2. The user must guess the correct number.
3. The system will tell if the guess is:
   - **Too High**
   - **Too Low**
4. The user has a maximum of **7 attempts per round**.
5. If the user guesses correctly, points are awarded based on attempts.
6. The user can play **multiple rounds**.

---

## Scoring System

| Attempts | Score |
|--------|--------|
| 1 – 2 attempts | 100 points |
| 3 – 4 attempts | 70 points |
| 5 – 6 attempts | 50 points |
| 7 attempts | 30 points |
